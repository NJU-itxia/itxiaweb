## 配置和编辑方法

先将此项目clone到本地, 在本地写好你要发布的md文件, 把你要发布的md文档放到你要发布的对应目录下, 将该文件名按summary.md中的格式添加到SUMMARY.md中的对应目录下, 修改README.md, 与SUMMARY.md同理, push到github上即可
